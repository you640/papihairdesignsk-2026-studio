# **App Name**: PAPI Hair Studio PWA

## Core Features:

- Appointment Booking: Multi-step form for booking appointments (service, stylist, date, time).
- Stylist Profiles: Display stylist profiles with portfolios and bios.
- Gallery: Showcase a portfolio of work with images uploaded to Firebase Storage.
- Online Shop: E-commerce functionality for products with cart and checkout (order form, no payment gateway).
- User Authentication: Enable users to log in/sign up using email/password and Google.
- Admin Dashboard: CRUD functionality for services, stylists, products, gallery, and blog posts.  Admin user management.
- Loyalty Program: Track user loyalty points and rewards

## Style Guidelines:

- Primary color: A warm, inviting coral (#FF7F50) evoking a sense of beauty and care.
- Background color: Light beige (#F5F5DC), providing a neutral and soft backdrop.
- Accent color: Muted gold (#D4AF37) for highlights and calls to action, adding a touch of luxury.
- Headline font: 'Playfair', serif, for titles and headings to convey elegance.
- Body font: 'PT Sans', sans-serif, for body text, ensuring readability and a modern feel. Using 'Playfair' for titles and headings.
- Use elegant, minimalist icons for navigation and actions.
- Subtle transitions and animations to enhance user experience, especially during booking and product interactions.  Framer Motion will handle the animations.
